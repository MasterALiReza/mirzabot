# 🇬🇧 Ox Panel & OxBot (English)

OxBot is an automated solution for selling VPN services directly via a Telegram bot with a beautiful Web Admin Panel. It integrates with **Marzban**, **3x-ui**, **Alireza Panel**, **IBSng**, etc.

---

## ⚙️ Features

### 🖥️ Web Admin Panel
- **Dashboard**: View real-time sales, user counts, and server status.
- **Servers**: Connect and monitor multiple panels (Marzban, 3x-ui) in one place.
- **Products**: Create VPN packages (volume/time limits) and set prices.
- **Gateways**: Configure crypto (NowPayments) and local fiat payments.
- **Bot Customization**: Change bot texts and buttons directly from the web panel.

### 📱 Telegram Bot
- **Auto Delivery**: Instantly sends the config link and QR code to the user after payment.
- **Account Actions**: Users can check status, renew, buy extra volume, and get help.
- **Free Trial**: Offer automatic limited trial accounts to new users.
- **Anti-Abuse**: Require phone number verification to prevent multiple trial accounts.
- **Support**: Users can open tickets, and admins can reply to them from the panel.

## 🚀 Installation & Update

### Prerequisites
- Ubuntu Server 22.04+ (Fresh server recommended)
- A domain pointed to your server IP.

### 🔧 Installation
Run this command in your root terminal:
```bash
curl -o install.sh -L https://raw.githubusercontent.com/MasterALiReza/OxBot/main/install.sh && bash install.sh
```
*Select option **1** to install.*

### 🔄 Update
Run the same command and select option **2 (Update OxBot)**:
```bash
curl -o install.sh -L https://raw.githubusercontent.com/MasterALiReza/OxBot/main/install.sh && bash install.sh
```
