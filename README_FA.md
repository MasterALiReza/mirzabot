# 🇮🇷 اکس پنل و اکس بات (Persian)

اکس‌بات (OxBot) یک سیستم هوشمند و خودکار برای فروش اکانت‌های VPN از طریق ربات تلگرام به همراه پنل وب مدیریت است. این ربات از پنل‌های **مرزبان (Marzban)**، **3x-ui**، **علیرضا پنل**، **IBSng** و... پشتیبانی می‌کند.

---

## ⚙️ ویژگی‌ها و امکانات

### 🖥️ پنل مدیریت تحت وب (Web Panel)
- **داشبورد آماری:** مشاهده گزارش فروش، آمار کاربران زنده ربات و وضعیت سرورها.
- **مدیریت سرورها:** متصل کردن چندین سرور و پنل مختلف در یک بخش واحد.
- **مدیریت محصولات:** تعریف بسته‌های اشتراکی (با محدودیت حجم و زمان) و قیمت‌گذاری آن‌ها.
- **درگاه‌های پرداخت:** راه‌اندازی آسان درگاه ریالی و درگاه کریپتو (نوپی‌منتز).
- **تنظیمات ربات:** شخصی‌سازی پیام‌ها و دکمه‌های ربات تلگرام بدون نیاز به کدنویسی.

### 📱 ربات تلگرام (Telegram Bot)
- **تحویل آنی:** ساخت خودکار اکانت و ارسال فوری کانفیگ و کد QR به محض پرداخت کاربر.
- **مدیریت سرویس:** امکان تمدید، خرید حجم اضافه و دریافت مجدد اشتراک‌ها توسط کاربر.
- **تست رایگان:** تحویل اکانت تست خودکار به کاربران جدید.
- **ضدسوءاستفاده:** تایید شماره تلفن برای جلوگیری از دریافت مکرر اکانت تست.
- **پشتیبانی:** امکان ارسال تیکت توسط کاربر و پاسخ مستقیم ادمین از طریق پنل وب.

## 🚀 نصب و بروزرسانی

### پیش‌نیازها
- سیستم‌عامل اوبونتو سرور 22.04 به بالا (ترجیحاً سرور خام)
- یک دامنه یا ساب‌دامنه متصل به IP سرور.

### 🔧 دستور نصب مستقیم
دستور زیر را در ترمینال سرور خود (دسترسی root) وارد کنید:
```bash
curl -o install.sh -L https://raw.githubusercontent.com/MasterALiReza/OxBot/main/install.sh && bash install.sh
```
*سپس گزینه **۱** را جهت نصب انتخاب کنید.*

### 🔄 بروزرسانی ربات
جهت آپدیت به آخرین نسخه بدون پاک شدن اطلاعات قبلی، دستور زیر را اجرا کرده و گزینه **۲ (Update OxBot)** را انتخاب کنید:
```bash
curl -o install.sh -L https://raw.githubusercontent.com/MasterALiReza/OxBot/main/install.sh && bash install.sh
```
