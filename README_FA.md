# 🤖 اکس پنل و اکس بات (Ox Panel)

🇬🇧 **[Click here for English Documentation (README.md)](README.md)**

---

یک سیستم هوشمند و خودکار برای فروش و مدیریت اشتراک‌های VPN از طریق ربات تلگرام، به همراه پنل مدیریت تحت وب بسیار زیبا و مدرن.

---

## 🔌 پنل‌های تحت پشتیبانی

OxBot قابلیت اتصال مستقیم و ساخت خودکار اکانت برای پنل‌های زیر را دارد:
- 🟢 **مرزبان (Marzban)**
- 🟢 **3x-ui** / **X-UI**
- 🟢 **هیدیفای (Hiddify)**
- 🟢 **علیرضا پنل (Alireza Panel)**
- 🟢 **IBSng**
- 🟢 **WG-Dashboard** (وایرگارد)
- 🟢 **میکروتیک (Mikrotik)**

---

## ⚙️ ویژگی‌ها و امکانات

### 🖥️ پنل مدیریت تحت وب (Web Panel)
- **داشبورد گزارشات:** مشاهده آمار فروش، تعداد کاربران زنده ربات و نمودارهای سود روزانه/ماهانه.
- **مدیریت سرورها:** متصل کردن و پایش وضعیت چندین سرور و پنل مختلف به صورت یکجا.
- **محصولات و دسته‌بندی:** ایجاد بسته‌های اشتراکی (زمانی و حجمی)، قیمت‌گذاری و مدیریت دسته‌ها.
- **درگاه‌های پرداخت:** راه‌اندازی درگاه پرداخت کریپتو (NowPayments) و درگاه‌های ریالی محلی.
- **تنظیمات ربات:** شخصی‌سازی و تغییر لحظه‌ای پیام‌ها، منوها و دکمه‌های ربات تلگرام.

### 📱 ربات تلگرام (Telegram Bot)
- **تحویل آنی اکانت:** ساخت اتوماتیک اکانت و ارسال فوری مشخصات اتصال و کد QR بلافاصله پس از پرداخت.
- **خدمات اشتراک:** بررسی وضعیت اکانت، تمدید اشتراک، خرید حجم اضافه و دریافت مجدد کانفیگ توسط کاربر.
- **تست رایگان خودکار:** ارائه اشتراک تست زمان‌دار/حجم‌دار به کاربران جدید جهت بررسی سرعت.
- **سیستم ضد اسپم:** احراز هویت با شماره تلفن برای جلوگیری از دریافت مکرر اکانت تست.
- **پشتیبانی تیکتی:** ارسال تیکت و گزارش باگ توسط کاربر و امکان پاسخ مستقیم ادمین از داخل پنل وب.

## 🚀 نصب و بروزرسانی

### پیش‌نیازها
- **سیستم‌عامل:** اوبونتو سرور 22.04 به بالا (ترجیحاً سرور خام)
- **دامنه:** دامنه یا ساب‌دامنه متصل به IP سرور شما.

### 🔧 دستور نصب مستقیم (نسخه پایدار)
دستور زیر را در ترمینال سرور خود (دسترسی root) وارد کنید:
```bash
curl -o install.sh -L https://raw.githubusercontent.com/MasterALiReza/OxBot/main/install.sh && bash install.sh
```
*سپس گزینه **۱** را انتخاب کنید.*

### 🔄 بروزرسانی ربات
جهت آپدیت به آخرین نسخه بدون پاک شدن اطلاعات قبلی، دستور زیر را اجرا کرده و گزینه **۲ (Update OxBot)** را انتخاب کنید:
```bash
curl -o install.sh -L https://raw.githubusercontent.com/MasterALiReza/OxBot/main/install.sh && bash install.sh
```
